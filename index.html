Article Design Game

Welcome to Article Design Game!

Welcome to Article Design Game! This online web game promotes learning about different types of graphic design in an interactive space by encouraging users to create different types of articles. When starting the game, users start from scratch using a grid format. There will also be multiple example designs for the user to look at and gain inspiration from.


To begin, select one item from the menu bar to start arranging your page!
